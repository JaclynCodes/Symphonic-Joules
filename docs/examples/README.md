## 📋 Example Template

When creating new examples, use this structure:

```
example-name/
├── README.md           # Description and instructions
├── example.py          # Main example code
├── requirements.txt    # Dependencies (if any)
├── data/              # Sample data files
└── output/            # Expected output files
```
